# FernAir - Premium HVAC Solutions

FernAir is a premium Air Conditioning (AC) service website catering to the Newcastle and Hunter Valley regions. This website is designed with a modern aesthetic, high performance, and intuitive navigation using **Vite** and **Tailwind CSS**.

## ✨ Key Features

- **Modern & Responsive Design**: A professional look that adapts seamlessly to mobile, tablet, and desktop devices.
- **Fast Navigation**: Smooth transitions between Home, Services, Emergency, and Quote pages.
- **Interactive Forms**: An elegant and functional system for requesting price estimates.
- **24/7 Emergency Mode**: A dedicated page for emergency services with quick contact access.
- **Performance Optimized**: Powered by Vite for a lightning-fast development experience and page load times.

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Modern utility-first design system).
- **Tooling**: [Vite](https://vitejs.dev/) (Next Generation Frontend Tooling).
- **Icons**: Google Material Symbols.
- **Typography**: Inter Font Family.

## 🛠️ Local Setup Instructions

Follow these steps to run the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/username/fernair-base.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd fernair-base
   ```

3. **Install dependencies:**
   Ensure you have [Node.js](https://nodejs.org/) installed. First, enter the `web` folder:
   ```bash
   cd web
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in your browser:**
   Go to `http://localhost:5173` to view the website.
